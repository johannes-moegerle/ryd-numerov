property mu: float

Return the reduced mass in atomic units, i.e. return m_{Core} / (m_{Core} + m_e).

To get the reduced mass in atomic units, we use the species dependent Rydberg constant

\[R_{m_{Core}} / R_{\infty} = \frac{m_{Core}}{m_{Core} + m_e}\]
